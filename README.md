# JS Calculator
This project is a basic calculator maked with web technologies using the framework Electron JS for do it look like an actually desktop application. In this application, you can do all of basics operations like a common calculator: sum `+`, subtract `-`, multiplication `√ó` and division `√∑`. Also you can do some intermedium operations using the parentheses `( )` through the keyboard keys.
